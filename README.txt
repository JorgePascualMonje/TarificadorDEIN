PROYECTO TARIFICADOR JORGE PASCUAL MONJE

El siguiente proyecto se ha realizado en un jframe utilizando java swing.

Al iniciar el programa se abrirá una pestaña con 4 variables: destino, Tipo, Modalidad y Peso, separados de estos
cuatro elementos hay un botónj para calcular el precio final de nuestro envió así como una lína de texto que nos
indica dicho texto.

El destino solo puede ser indicado como Nacional, el tipo de envío puede ser indicado como Carta o Tarjeta Postal,
la Modalidad puede ser indicada como  Ordinaria, Certificada, Urgente, y Urgente Certificada y el peso debe ser
indicado manualmente.

Si la Modalidad es indicada como Certificada o Urgete Certificada, dos nuevas variables aparecen, Valor Declarado y
Acuse de recibo, esto se ha diseñado así para que el usuario no se sienta confuso al llenenar un envió no 
cerfificado. La modalidad también afecta a la lista de precios que debe usar el programa.

El peso determina el precio exacto del envío, dependiendo de la modalidad, si el peso es demasiado alto el progama
se lo indicará al usuario.

Valor Declarado aumentará el precio del envío por cada 50€ declarados, si el valor declarado es menor de 50 no se
añadirá ningún coste adicional.

Si la casilla de Acuse de Recibo está marcada, se añadirá un total de 1.94€ al precio final.

Una vez se pulsa el botón de Calcular, el programa mostrará el precio final en pantalla.
